# About

This is a sample CRUD project build upon:

* Spring Boot
* React.js
* MongoDB

# Deploy

Use below command to deploy the app:

```
docker-compose up
```

Once the application is up, open following URL:

http://localhost:3000

### API endpoints:

http://localhost:8080
